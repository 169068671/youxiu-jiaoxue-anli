# 代码造物 - STEAM与FreeCAD数字工坊

> "在比特与原子的交界处，用代码定义万物。"

## 项目简介

本项目是江苏省东台中学通用技术校本课程——《STEAM 理念下基于编程建模的高中 3D打印校本课程的开发与研究》（盐城市教育学会课题，编号：2024B-178）的配套网站。

课程将 **STEAM**（科学、技术、工程、艺术、数学）理念与 **FreeCAD Python 编程**深度融合，让学生化身为数字造物主，用代码生成复杂的几何形态，学习使用 **Qt GUI** 开发交互面板，最终将数字创意通过 3D 打印机转化为真实物理产品。

## 课程结构（20课时）

| 模块 | 主题 | 课时 | 核心内容 |
|:---:|:---|:---:|:---|
| **模块一** | 🚀 启航 —— STEAM与数字造物 | 2课时 | 3D打印原理、FreeCAD基础 |
| **模块二** | 🥉 青铜时代 —— Python与三维空间 | 4课时 | 变量、循环、条件判断 |
| **模块三** | 🥈 白银时代 —— 数学之美与算法 | 4课时 | 列表、函数、三角函数、分形 |
| **模块四** | 🥇 黄金时代 —— 掌控交互 | 4课时 | GUI开发、按钮、滑动条、布局 |
| **模块五** | 💎 铂金时代 —— 动力与模拟 | 3课时 | 定时器、碰撞检测、轨迹追踪 |
| **模块六** | 👑 王者时代 —— 终极挑战 | 3课时 | 项目设计、3D打印、答辩展示 |

## 网站特点

- ✅ 完整的20节课内容
- ✅ 标准化的"物理/数理原理 + FreeCAD 脚本 + 实验报告 + 制作指南"教学模式
- ✅ 代码高亮与清晰排版
- ✅ 响应式设计，支持移动端访问
- ✅ 独立课程目录与逐课导航
- ✅ 可离线直接打开，不依赖网络和本地服务器

## 网站形式

- **发布形式**：静态HTML
- **课程源文件**：Markdown
- **样式文件**：`docs/site.css`
- **网络要求**：无

## 打开方法

1. 解压网站压缩包，并保持文件夹结构不变。
2. 进入 `docs` 文件夹。
3. 双击 `index.html` 即可浏览。

不要只复制单个 `index.html`；课程页面、附录和 `site.css` 需要与它保持原有相对位置。
