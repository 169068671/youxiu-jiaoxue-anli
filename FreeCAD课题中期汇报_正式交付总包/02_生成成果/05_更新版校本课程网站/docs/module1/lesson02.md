## 先把关系说成人话

**数量关系：** 球体大小由半径决定；直径 = 2 × 半径

**怎么理解：** 半径是从球心到表面的距离，直径穿过球心。

**数字例子：** 半径10毫米的球，直径就是20毫米。

**生活类比：** FreeCAD中的Shape不是图片，而是可计算的几何体。

## 已验证的课堂最小代码

下面的代码聚焦本课核心知识点，适用于带 Python 控制台的 FreeCAD 桌面版。

```python
import FreeCAD as App
import Part

doc = App.newDocument("Lesson02")
sphere = doc.addObject("Part::Feature", "Sphere")
sphere.Shape = Part.makeSphere(10)
doc.recompute()
```

## 按这五步运行

1. 打开FreeCAD，新建文档。
2. 打开“视图 → 面板 → Python控制台”。
3. 先原样运行代码，确认出现半径10毫米的球体。
4. 把 `10` 改成 `15`，重新运行并比较球体大小。
5. 记录“改了什么、结果怎样、为什么会变”。
