## 💻 快速开始

### 第一步：环境准备

1. **下载并安装 FreeCAD**（推荐 0.21+ 版本）
   - 官网：https://www.freecad.org/

2. **启动 FreeCAD，打开 Python 控制台**
   - 点击菜单栏 `视图 (View)` → `面板 (Panels)` → 勾选 `Python 控制台 (Python console)`

### 第二步：写下你的第一行"创世代码"

```python
import FreeCAD as App
import Part

# 创造一个长宽高都是10的立方体
my_box = Part.makeBox(10, 10, 10)

# 把立方体显示在3D视图中
Part.show(my_box)

# 刷新视图
App.ActiveDocument.recompute()
```

🎉 **魔法生效**：屏幕上出现了一个正方体！这就是你的"Hello 3D World"！
