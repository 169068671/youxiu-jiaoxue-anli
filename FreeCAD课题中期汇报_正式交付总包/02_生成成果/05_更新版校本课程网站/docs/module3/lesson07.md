2. 看懂并计算：空间点 =（X坐标，Y坐标，Z坐标）
3. 运行一段最小代码，完成：连接4个点形成折线

## 先把关系说成人话

**数量关系：** 空间点 =（X坐标，Y坐标，Z坐标）

**怎么理解：** 先写点，再把相邻点依次连接，就得到多段线。

**数字例子：** (0,0,0)→(20,0,0)→(20,20,0)形成一个直角。

**生活类比：** 列表像路线表，每一个点都是下一站。

## 已验证的课堂最小代码

下面的代码聚焦本课核心知识点，适用于带 Python 控制台的 FreeCAD 桌面版。

```python
import FreeCAD as App
import Part

doc = App.newDocument("Lesson07")
points = [App.Vector(0,0,0), App.Vector(20,0,0),
          App.Vector(20,20,0), App.Vector(40,20,10)]
line = doc.addObject("Part::Feature", "Line")
line.Shape = Part.makePolygon(points)
doc.recompute()
```
