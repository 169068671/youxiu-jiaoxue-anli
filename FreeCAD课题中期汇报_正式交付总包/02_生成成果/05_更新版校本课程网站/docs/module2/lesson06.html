代码造物数字工坊

📍 第 6 课：逻辑岔路口（if 条件判断与间隔挖空）

条件判断让程序在不同情况下选择不同动作。

本课要完成什么

  1. 用自己的话说清楚：条件判断让程序在不同情况下选择不同动作。
  2. 看懂并计算：余数 = 序号除以2后剩下的数
  3. 运行一段最小代码，完成：只显示偶数位置

先把关系说成人话

数量关系： 余数 = 序号除以2后剩下的数

怎么理解： 余数为0是偶数位置，余数为1是奇数位置。

数字例子： 序号0、2、4、6、8显示；1、3、5、7、9留空。

生活类比： if像岔路口，条件决定走哪一条路。

已验证的课堂最小代码

下面的代码聚焦本课核心知识点，适用于带 Python 控制台的 FreeCAD 桌面版。

import FreeCAD as App
import Part

doc = App.newDocument("Lesson06")
for i in range(6):
    if i % 2 == 0:
        box = doc.addObject("Part::Feature", "Box")
        box.Shape = Part.makeBox(8, 8, 8)
        box.Placement.Base.x = i * 12
doc.recompute()

按这五步运行

  1. 打开FreeCAD，新建文档。
  2. 打开“视图 → 面板 → Python控制台”。
  3. 先原样运行代码，确认出现模型、控件或输出。
  4. 一次只修改一个数字，再运行一次。
  5. 记录“改了什么、结果怎样、为什么会变”。
