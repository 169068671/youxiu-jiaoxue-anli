# 📍 第 4 课：参数化思维（空间位置的向量控制）

> 向量同时记录方向和距离，是三维空间中的移动指令。

## 本课要完成什么

1. 用自己的话说清楚：向量同时记录方向和距离，是三维空间中的移动指令。
2. 看懂并计算：新位置 = 原位置 + 位移向量
3. 运行一段最小代码，完成：沿X移动50毫米

## 先把关系说成人话

**数量关系：** 新位置 = 原位置 + 位移向量

**怎么理解：** Vector(50,0,0)表示沿X方向移动50毫米。

**数字例子：** 原位置(0,0,0)，加上(50,0,0)，新位置是(50,0,0)。

**生活类比：** X是左右，Y是前后，Z是上下。

## 参数到底在哪里

参数就是**可以提前设定、反复修改，并决定模型结果的数据**。本课把参数集中放在代码最上方：

| 参数 | 含义 | 初始值 |
|---|---|---:|
| `length` | 立方体沿 X 方向的长度 | 20 mm |
| `width` | 立方体沿 Y 方向的宽度 | 20 mm |
| `height` | 立方体沿 Z 方向的高度 | 20 mm |
| `move_x` | 沿 X 方向移动的距离 | 50 mm |
| `move_y` | 沿 Y 方向移动的距离 | 0 mm |
| `move_z` | 沿 Z 方向移动的距离 | 0 mm |

只修改参数区中的数字，下面的建模语句不用重写，模型的尺寸和位置就会自动改变。这就是最基础的**参数化建模**。

## 已验证的课堂最小代码

下面的代码聚焦本课核心知识点，适用于带 Python 控制台的 FreeCAD 桌面版。

```python
import FreeCAD as App
import Part

# ===== 参数区：主要修改这里 =====
length = 20       # 长度，单位：mm
width = 20        # 宽度，单位：mm
height = 20       # 高度，单位：mm

move_x = 50       # 沿 X 方向移动距离
move_y = 0        # 沿 Y 方向移动距离
move_z = 0        # 沿 Z 方向移动距离
# ==============================

doc = App.newDocument("Lesson04")
box = doc.addObject("Part::Feature", "MovedBox")
box.Label = "参数化移动方块"

# 建模区：尺寸和位置都引用上面的参数
box.Shape = Part.makeBox(length, width, height)
box.Placement.Base = App.Vector(move_x, move_y, move_z)

doc.recompute()
Gui.activeDocument().activeView().viewAxonometric()
Gui.activeDocument().activeView().fitAll()
```

## 做三次参数实验

每次只修改参数区，然后重新运行代码并比较模型位置：

| 实验 | 参数设置 | 看到的结果 |
|---|---|---|
| 1：沿 X 移动 | `move_x = 50, move_y = 0, move_z = 0` | 方块向右移动 50 mm |
| 2：沿 Z 上升 | `move_x = 0, move_y = 0, move_z = 30` | 方块向上移动 30 mm |
| 3：斜向移动 | `move_x = 40, move_y = 30, move_z = 0` | 方块同时沿 X、Y 移动 |

再把 `length` 从 `20` 改为 `40`。如果方块变长，就说明你已经同时控制了**尺寸参数**和**位置参数**。
