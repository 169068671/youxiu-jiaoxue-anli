# 📍 第 3 课：控制台的魔法（Python变量的引入）

> 变量是带名字的数据盒子。改盒子里的数，模型尺寸随之改变。

## 本课要完成什么

1. 用自己的话说清楚：变量是带名字的数据盒子。改盒子里的数，模型尺寸随之改变。
2. 看懂并计算：圆柱直径 = 2 × 半径
3. 运行一段最小代码，完成：创建半径20、高30的圆柱

## 先把关系说成人话

**数量关系：** 圆柱直径 = 2 × 半径

**怎么理解：** radius保存半径，height保存高度，代码用变量创建圆柱。

**数字例子：** radius=20，height=30：圆柱直径40毫米，高30毫米。

**生活类比：** 变量名像标签，数值像盒子里的物品。

## 已验证的课堂最小代码

下面的代码聚焦本课核心知识点，适用于带 Python 控制台的 FreeCAD 桌面版。

```python
import FreeCAD as App
import Part

radius = 20
height = 30
doc = App.newDocument("Lesson03")
cylinder = doc.addObject("Part::Feature", "Cylinder")
cylinder.Shape = Part.makeCylinder(radius, height)
doc.recompute()
```

## 按这五步运行

1. 打开FreeCAD，新建文档。
2. 打开“视图 → 面板 → Python控制台”。
3. 先原样运行代码，确认出现半径20毫米、高30毫米的圆柱。
4. 一次只修改一个数字，再运行一次。
5. 记录“改了什么、结果怎样、为什么会变”。
