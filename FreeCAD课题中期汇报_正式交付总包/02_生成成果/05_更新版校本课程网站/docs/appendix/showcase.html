案例一：Python参数化可变形收纳盒

作者：张同学（高二3班）

项目简介： 通过滑动条改参数，隔板自动增加。可以自定义收纳盒的格子数量和大小，适合存放不同尺寸的物品。

核心功能： - 滑动条控制格子数量（1-9格） - 自动计算每个格子的大小 - 壁厚可调 - 一键导出STL

技术亮点： - 使用 QSlider 实现实时参数调节 - 嵌套循环生成格子阵列 - 布尔运算合并所有部件

案例三：数学之美——3D莫比乌斯环

作者：王同学（高二1班）

项目简介： 用 math.sin 和 math.cos 绘制3D莫比乌斯环和螺旋线，展示数学之美。

核心功能： - 参数化莫比乌斯环 - 可调节环的半径和扭转次数 - 彩色渐变效果

技术亮点： - 参数方程实现莫比乌斯环 - B样条曲线拟合平滑曲面 - 颜色渐变算法

数学公式：

x = (R + r*cos(t/2)) * cos(t)
y = (R + r*cos(t/2)) * sin(t)
z = r * sin(t/2)
