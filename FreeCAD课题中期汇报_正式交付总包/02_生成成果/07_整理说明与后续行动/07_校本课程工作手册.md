# 江苏省东台中学校本选修课管理制度  

# 第一章总则  

第一条为丰富学生的课外生活，拓宽学生的知识面，培养学生的综合素质，促进学生的个性发展，提升学校的办学水平，学校设立校本选修课程。为规范校本选修课程管理，依据《教育法》、《普通高中课程标准》等法规文件，结合本校实际，特制定本管理制度。  

第二条校本选修课程是学校课程体系的重要组成部分，是在学生完成必修课程学习任务的前提下，根据自身兴趣和特长自愿选择的课程，选修课程不计入学生学业考核。  

第三条选修课程学时应当符合学校的课程设置和教学计划，学校应合理配置教师资源、教学设施和教学条件，保证选修课程的开展。  

第四条选修课程设置应当充分考虑学生的兴趣和需求，突出专业特长和特色，促进学生的全面发展。  

第五条学校应当组织选修课程教师对选修课程进行相应的教学研讨和培训，提高教师教学水平和教研能力。  

# 第二章选课  

第六条学校应当每学期期初公布校本选修课程目录，包括课程名称、授课教师、课程简介、选课对象、选课人数限制等信息。  

第七条 学校应当根据学生的需求和实际情况制定选修课程开课计划和教学安排。  

第八条学生根据自身兴趣和发展需求，在规定时间内通过学校选课系统进行选课。每位学生每学期可选修1门校本选修课程。  

第九条学生选课以个人自愿为原则，学生可以根据自己的兴趣和特长自主选择，不得强迫学生选择特定的选修课程。  

第十条学生选择的选修课程必须符合学生的学业规划，不得影响学生的必修课程学习和教学秩序。  

第十一条选课结束后，学校根据选课情况进行调整，确定最终开课课程和学生名单，并通过校园网等方式公布。学生选课后如需调整选修课程，需经过班主任同意、年级批准并报教务处审批。  

# 第三章教学管理  

第十二条校本选修课程教学内容应科学、规范，符合学生认知规律，注重实践性和趣味性，避免与必修课程内容重复。  

第十三条校本选修课程教学形式应灵活多样，可采用讲座、研讨、实践、考察等多种形式，  

激发学生学习兴趣，提高教学效果。  

第十四条学校应当建立选修课程档案，对学生的选修课程情况进行详细记录，及时掌握学生的学习状况。  

第十五条校本选修课程授课教师应具备相应的专业知识和教学能力，并认真备课、上课，保证教学质量。对于选修课程教师的任课安排、教学质量等情况，学校应当进行定期评估、并进行适当的考核评价。  

# 第四章安全管理  

第十六条学校应当加强选修课程场所的安全管理，建立安全管理制度、安全预案和规范流程，落实安全责任，确保学生在选修课程学习中的安全。  

第十七条学校应当对选修课程提供的实验室、设备和材料等进行定期维护和检修，保障选修课程教学的正常进行。  

第十八条学校应当建立选修课程学生考勤制度，对学生选修课程的出勤情况进行统计和管理。  

第十九条校本选修课程授课教师应加强课堂管理，维护课堂秩序，防止发生安全事故。  

# 第五章监督管理  

第二十条学校应当成立校本选修课程管理领导小组，负责校本选修课程的规划、开发、实施、评价等工作。  

第二十一条学校应当建立选修课程学生的评价和管理制度，对学生的选修课程学习情况进行评估，对优秀学生给予奖励，对学习不认真、违规违纪的学生进行规范和处理。  

第二十二条学校应当建立选修课程的评估制度，对选修课程的教学质量和教学效果进行评价，及时调整和完善选修课程的设置和开展。  

第二十三条学校鼓励教师积极参与校本选修课程的开发和实施，并对优秀课程和教师进行表彰奖励。  

# 第六章附则  

第二十四条学校应当建立选修课程管理制度宣传教育工作，向学生宣传选修课程的意义和重要性，引导学生正确选择选修课程。  

第二十五条学校应当根据教育部门的要求，定期向教育主管部门上报选修课程的开展情况。  

第二十六条本规定自发布之日起正式施行，如有调整和变更，另行发布。
