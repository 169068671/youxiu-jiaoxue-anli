## 二、20 课时课程体系

| 模块 | 主题 | 课时 | 核心内容 |
|---|---|---:|---|
| 模块一 | STEAM 与数字造物 | 2 | 3D 打印原理、FreeCAD 基础 |
| 模块二 | Python 与三维空间 | 4 | 变量、循环、条件判断 |
| 模块三 | 数学之美与算法 | 4 | 列表、函数、三角函数、分形 |
| 模块四 | GUI 交互开发 | 4 | 按钮、滑动条、文本输入、布局 |
| 模块五 | 动力与模拟 | 3 | 定时器、碰撞检测、轨迹追踪 |
| 模块六 | 综合设计与制造 | 3 | 项目设计、切片打印、答辩展示 |

课堂 PPT 与课程网站采用相同的精简代码和中文数量关系。每课包含知识讲解、代码分段、预测与观察、纠错、分层任务、作品评价和离堂检查。

## 三、快速使用

### 查看优秀教学案例

优先阅读申报案例 PDF；需要修改时使用 `word版` 中的 DOCX/XLSX；课堂实施时配合“发现问题 教案以及资源”中的教学设计、FCStd、STEP 和示例图。

### 使用课堂 PPT

进入 `FreeCAD课题中期汇报_正式交付总包/02_生成成果/03_校本课程课堂PPT_20课时精讲版/`，按课次顺序打开。课程代码已按逐课最小示例整理，并在对应 FreeCAD 环境中进行过验证。

### 打开课程研究数据平台

打开 `02_生成成果/04_课程研究数据平台/index.html`。示例 CSV 仅用于演示导入格式和测试流程，不代表真实科研结论。

### 浏览课程网站

进入 `02_生成成果/05_更新版校本课程网站/docs/`，双击 `index.html`。请保持 `docs` 目录结构完整，不要单独移动首页文件。

## 四、文件格式与软件

- `PDF`：正式阅读、打印或报送；
- `DOC/DOCX`：Microsoft Word 或兼容办公软件编辑；
- `XLSX/CSV/ET`：Excel、WPS 表格等软件处理；
- `PPTX`：PowerPoint 或兼容演示软件播放；
- `FCStd`：使用 FreeCAD 打开和编辑；
- `STEP`：用于 CAD 交换、检查或 3D 打印前处理；
- `HTML/MD/CSS/JS`：课程网站和研究数据平台源文件；
- `ZIP`：历史网站或原始资源归档。
